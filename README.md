# Inventory Automation System – AUBO Robotics

This project is a lightweight inventory tracking web app designed to simplify and automate hardware inventory management.  
The goal was to replace manual spreadsheet-based tracking with a more intuitive, easy-to-use interface that allows users to quickly view, update, and monitor inventory levels.

## Overview

Managing inventory with many part numbers and hardware variations can become error-prone when done manually.  
This tool provides a simple web-based interface for tracking inventory items, making updates, and identifying low or inconsistent stock.

The system was designed with a focus on:

- Usability for technical and non-technical users  
- Clear visibility of inventory status  
- Safe and simple updates  
- Ability to scale to many parts and variations  
- Low-cost, low-code implementation  

## Features

- View inventory items in a structured table
- Track part numbers and hardware variations
- Update inventory levels
- Highlight low inventory
- Simple UI designed for quick lookup
- Lightweight static site deployment

## Tech Stack

- HTML / CSS / JavaScript
- GitHub Pages (deployment)
- Low-code / AI-assisted development (Claude AI)
